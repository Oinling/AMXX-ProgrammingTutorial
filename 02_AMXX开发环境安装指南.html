CS1.6 AMXX开发环境安装指南

版本选择说明

根据您的CS1.6版本选择安装流程：

  • 旧版CS：Exe build ≤ 9000，可使用低版本或高版本Metamod和AMXModX
  • 新版CS：Exe build > 9000，必须使用ReHLDS、ReGameDLL等组件配合高版本AMXModX

检查方法：游戏中按`打开控制台，输入version，查看"Exe build"末尾括号内数字。

1. 基础准备(显示文件扩展名)

在Windows设置中启用"显示文件扩展名"功能。

2. 旧版CS安装流程 (Exe build ≤ 9000)

2.1 安装Metamod

下载：Metamod 1.21.1

解压后将addons文件夹复制到cstrike目录

2.2 修改liblist.gam

编辑cstrike/liblist.gam文件，修改以下内容：

gamedll "addons/metamod/dlls/metamod.dll" gamedll_linux "addons/metamod/dlls/metamod.so" gamedll_osx "addons/metamod/dlls/metamod.dylib"

2.3 创建plugins.ini

在cstrike/addons/metamod/下创建plugins.ini文件，添加：

win32 addons/amxmodx/dlls/amxmodx_mm.dll

2.4 安装AMX Mod X

下载：AMX Mod X 1.8.2和AMX Mod X 1.8.2 CS拓展模块

解压后将addons文件夹合并到cstrike目录

其它版本（不需要安装GameData修复包）：

AMX Mod X 1.9.0

AMX Mod X 最新版

3. 新版CS安装流程 (Exe build > 9000)

3.1 安装ReHLDS

下载：ReHLDS/releases

解压，将bin/win32文件夹里的所有内容放入CS目录

3.2 安装ReGameDLL

下载：ReGameDLL_CS/releases

解压，将bin/win32文件夹里的所有内容放入CS目录

3.3 安装Metamod-R

下载：Metamod-R

解压，将addons文件夹放入cs1.6\cstrike

修改liblist.gam：

gamedll "addons\metamod\metamod.dll" gamedll_linux "addons/metamod/metamod_i386.so"

3.4 安装高版本AMX Mod X

下载：AMX Mod X 1.10.0.5467和AMX Mod X 1.10.0.5467 CS拓展模块

解压，将addons文件夹合并放入cs1.6\cstrike

其它版本：

AMX Mod X 最新版

AMX Mod X 1.9.0

3.5 创建plugins.ini

在cstrike/addons/metamod/下创建plugins.ini文件，添加：

win32 addons/amxmodx/dlls/amxmodx_mm.dll

3.6 安装GameData修复包

下载：GameData

将所有文件复制到cstrike\addons\amxmodx\data文件夹

4. 开发工具安装

4.1 文本编辑器

推荐使用：

  • Visual Studio Code
  • Notepad++
  • AMX Mod X Studio

注意：保存源码时使用UTF-8无BOM编码

4.2 插件相关目录结构

重要目录说明：

  • addons/amxmodx/scripting - 源码和编译器
  • addons/amxmodx/scripting/include - 头文件
  • addons/amxmodx/plugins - 编译后的插件
  • addons/amxmodx/logs - 插件运行日志
  • addons/amxmodx/configs - 配置文件
  • addons/amxmodx/data/lang - 插件文本翻译
  • addons/amxmodx/modules - 模块

scripting文件夹里的amxxpc.exe是编译器，可以附带各种参数，编译单个源码.将.sma文件拖到其图标上就会原地生成.amxx文件.

scripting文件夹里的compile.exe可以一次编译多个源码.将.sma文件拖到其图标上就会在scripting/compile文件夹生成.amxx文件.

直接运行compile.exe可以编译scripting文件夹里的所有源码.

编译好的.amxx文件需要放入plugins文件夹，并在configs/plugins.ini文件内另起一行写上.amxx文件全名.

像这样:

myplugin.amxx ; 简单介绍我的插件

5. 验证安装

启动游戏，按~打开控制台，发送以下命令:

  • amxx version - 检查AMXX版本
  • amxx plugins - 查看运行中的插件
  • setinfo lang cn - 设置中文
  • amx_language cn - 设置中文
  • amx_langmenu - 打开语言菜单，查看菜单文本是否为中文
