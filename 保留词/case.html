AMXXPawn case关键字

case可以在switch语句的代码块内定义不同的分支执行代码。

case的语法1：case 常量表达式: 语句块

case的语法2：case 值较小的常量表达式 .. 值较大的常量表达式: 语句块

重点：

  • 当switch的控制表达式返回值等于case的常量表达式时，则执行case的语句块
  • 若switch语句含有多个case分支，分支的常量表达式绝不能相等或相交（例如已存在case 2 .. 4:情况下，则不能有case 2:、case 3:、case 4:）
  • 与其它编程语言不同，语句块是case语法不可分割、不可省略的一部分（至少要保留空代码块{}或一条语句）
  • case的语句块内不需要且不能使用break跳出switch语句
  • case的语句块被执行完毕后，会自然退出switch语句，不会进入其它case分支（除非使用goto跳转语句）

示例：

switch (random(5)) { case 0: { server_print("[AMXX]进入分支1 (随机值等于0)"); } case 1: { server_print("[AMXX]进入分支2 (随机值等于1)"); } case 2 .. 4: { server_print("[AMXX]进入分支3 (随机值等于2、3或4)"); } }
