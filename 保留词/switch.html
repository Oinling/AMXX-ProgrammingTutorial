AMXXPawn switch关键字

switch语句是一种多分支选择结构，根据控制表达式的值选择不同的分支代码执行。

switch语句的语法：switch (控制表达式) { 任意数量的case分支或default分支 }

switch语句只能在函数体内使用。

当控制表达式的值满足其中一个分支的匹配条件，就会执行该分支的代码，其它分支不会执行。

示例：

// 在0~4中取随机数 switch (random(5)) { case 0: { server_print("[AMXX]进入分支1 (随机值等于0)"); } case 1: { server_print("[AMXX]进入分支2 (随机值等于1)"); } default: { server_print("[AMXX]进入分支3 (随机值等于2、3或4)"); } }
